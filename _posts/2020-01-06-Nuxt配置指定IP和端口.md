>Nuxt配置指定IP和端口


# 非常简单，不说废话。

只要加一个配置即可！

第1步，复制这个代码（端口可改）
```
 "config": {
    "nuxt": {
      "host": "0.0.0.0",
      "port": "8090"
    }
  },

```

第2步，找到根目录下的package.json，把代码粘贴进去。

这是全部的代码，我这里配置在了8~13行。
